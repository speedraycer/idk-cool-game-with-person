# 😈 H4CK3R TERMINAL — Prank App

A fake "hacker" app for pranking little cousins (ages 9–12). Green Matrix rain,
a scrolling terminal that "breaks into" phones, scary beeps, phone vibration,
and dramatic **ACCESS GRANTED** flashes. It's 100% fake — it doesn't touch any
real device, network, or account. It just *looks* like elite movie hacking.

It's a single file (`index.html`), so there's nothing to install and no app
store needed. It runs in any phone browser and can be added to the home screen
so it looks and opens like a real app.

## How to put it on your phone

**Easiest way (no computer needed):**
1. Open GitHub on your phone and find `index.html` in this repo.
2. Tap **Raw** to open the file by itself.
3. It runs right there in the browser. Tap once to start it (that turns the
   sound on).

**Make it look like a real app (add an icon to your home screen):**
- **iPhone (Safari):** tap the **Share** button → **Add to Home Screen**.
- **Android (Chrome):** tap the **⋮** menu → **Add to Home screen**.

Now there's a `>_` icon on the phone. Tapping it opens the app fullscreen —
no browser bars — so it really looks like a secret hacking tool. 🕶️
